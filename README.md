## Instruction on setting up and running the project

### Accessing our website
You can view our website through here: [Link to Website on Github-Pages](https://nikelora.github.io/v13_BuildingBlocs2025/).

### Get a copy of our website model
There are multiple ways to get a copy or download of our website model.

1. Github Desktop
You can click on the <> Code and open with github desktop to clone it.

2. Git Command
git clone [repository-url]
cd [repo-name]

3. Download
You can click on the <> Code and Download Zip to get a copy of our files.

### Verify File Structure
1. Ensure you have the following files in the root folder
    - index.html
    - Pages
    - js
    - Data
    - CSS
    - Assets

### Run the project

Multiple ways to launch it

1. From the folder: You can launch it from your folder by double clicking the index.html file
    - Note: Some features may not work properly due to launching without a server

2. Using Visual Studio Code
    - Ensure you have Live Server Extension on
    - Click on the Go Live Button on the footer of the Visual Studio Code
        - Note! If you do not see the Go Live Button, it means your Visual Studio Code doesn't have the Live Server Extension.

## Description of functionalities

### 1. Homepage
- Hero section with inspriational message
- Quick navigation to Information and Clinics Section
- Responsive layout with gradient background
- Clear call to action buttons

### 2. Information Page
- Talk about Mental Health, what it is about, signs of it and importance
- Easy to read layout with images and containers

### 3. About Us Page
- Contains information regarding who we are, our purpose and where our resources come from
- Responsive layout and includes highlighted container to stand out important message

### 4. Clinic Finder
- Includes a Search Box, allowing users to search clinics by name, address, or doctor
- Contain filters to allow by area (Central, East, North, West)
- Information about clinics are loaded from JSON file
- Display clinic details such as:
    - Address
    - Phone number
    - Website links (hyperlink onto the clinic names)
- Easy reset of search text via the cross icon
- Responsive layout
- User-friendly error handling when no clinics match the search

### 5. Interactive Chatbot
- Floating chat button on all pages
- Quick reply options for:
    - Mental health information
    - About SernityHub
    - Finding clinics
- Reset feature when closed

### 6. Responsive Navigation & Footer
- Fixed navigation bar with logo
- Consistent footer across all pages
- Social media links (Instagram, X/Twitter, Youtube)
- Contact Information
