Project Name: SerenityHub
Date: 12/12/2025

SerenityHub is a web-based platform designed to spread awareness about the importance of mental health and to provide users with information about available help services as well as guidance. Se

Languages Used: HTML, CSS, JavaScript
Designed and Brainstorming using Figma

Functionalities:

About Us
- Provides an overview of the organization's mission and goals
- Explains the purpose of the platform

Navigation Bar and Footer Area(s)
- Displays the website logo, also able to click on to go back to homepage
- Contains clickable links to pages like About Us, Information and Clinic
- Footer also displays extra social media icons that can direct users to the respectful social media links

Information Page
- Provides educational content related to mental health
- General Idea: Explains and highlights the importance of mental health

Interactive Chatbox
- Opens when the user clicks on the chat icon
- Displays predefined options
- Allows users to navigate to relevant sites for more accurate help

Clinics Page 
- Displays a searchable list of mental health clinics and their details including address, contact, doctors and website links
- Allows users to search clinics by name, address, or doctor
- Allows filtering by geographic area using checkboxes
- Loads clinic data from a JSON file
